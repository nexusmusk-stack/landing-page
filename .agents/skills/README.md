# Skills

This directory contains specialized templates and instructions for the agent to perform specific, repeatable tasks. Each skill file defines the input requirements, execution steps, and expected output for a common developer workflow.

## Available Skills
- [Workflow](./workflow.md): Quy trình phát triển phần mềm toàn diện (E2E Lifecycle).
- [Plan Task](./plan-task.md): Phân tích và tạo kế hoạch chi tiết (Vietnamese) trong thư mục docs.
- [Coding](./coding.md): Implementing features following project standards.
- [Reviewer](./reviewer.md): Analyzing changes for bugs and style violations.
- [Fix Review](./fix-review.md): Iterating on code based on feedback.
- [QC](./qc.md): Final validation of implemented features against quality standards.
