# Rule: UI Design Guidelines

All web application designs must feel "Premium" and "Dynamic".

## Core Guidelines
1. **Glassmorphism**: Use subtle background blurs and border-radii for components.
2. **Harmonious Palettes**: Avoid default browser colors. Use CSS variables for a curated theme.
3. **Typography**: Use modern sans-serif fonts (e.g., Inter, Roboto).
4. **Spacing**: Follow an 8-pixel grid system for consistency.
5. **Dark Mode**: Support dark mode by default with auto-switching or a toggle.

## Animation
- Use `framer-motion` (if using React) or standard CSS transitions for all interactions (hover, toggle, entrance).
- Keep animations subtle (e.g., 200ms - 400ms duration).
